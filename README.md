# Natural Deduction Solver

A simple natural deduction solver for propositional logic.
Built with TypeScript and Svelte, the tool aims to provide a comprehensive and easy-to-use interface for learning and solving natural deduction proofs.


## Dependencies
- [Svelte](https://svelte.dev/)
- [TypeScript](https://www.typescriptlang.org/)

## Installation and running
The project needs [Node.js](https://nodejs.org/) to run. After installing Node.js, clone the repository and run the following commands in the project directory:

```bash
npm install
npm run build
cd dist
npm run preview
```
